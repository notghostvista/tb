Términos de Servicio

Fecha de Vigencia: 28 de abril de 2025

Su privacidad es importante para nosotros. Estos Términos de Servicio explican cómo recopilamos, usamos y protegemos su información personal. Al utilizar nuestro sitio web, usted acepta los términos descritos a continuación.

1. Recopilación de Datos

  • Solo recopilamos datos personales cuando la ley lo exige, como en respuesta a solicitudes gubernamentales, citaciones u otras obligaciones legales.
  • No recopilamos información personal innecesaria. Solo se conserva la información mínima requerida para el cumplimiento.
  • No utilizamos sus datos para publicidad, elaboración de perfiles ni análisis no relacionados con el cumplimiento legal.

2. Uso de los Datos

  • Sus datos nunca se venden, alquilan ni comparten con terceros para fines de marketing o comerciales.
  • Cualquier dato recopilado se utiliza estrictamente para el cumplimiento legal y para cumplir con las obligaciones requeridas por las autoridades gubernamentales.
  • No utilizamos sus datos para ningún propósito que no sea el requerido por la ley.

3. Protección de Datos

  • Implementamos medidas de seguridad estándar de la industria para proteger sus datos contra el acceso, divulgación, alteración o destrucción no autorizados.
  • El acceso a sus datos está estrictamente limitado al personal autorizado que debe mantener la confidencialidad de la información.

4. Retención de Datos

  • Conservamos sus datos solo el tiempo necesario para cumplir con las obligaciones legales.
  • Una vez que los datos ya no sean requeridos por la ley, se eliminan o anonimizan de forma segura.

5. Prohibición de Uso Indebido de Datos

  • Tenemos una política de tolerancia cero para el uso indebido de datos personales. Cualquier uso indebido resultará en acciones estrictas, incluidas medidas disciplinarias para el personal y la notificación a las autoridades si es necesario.
  • Revisamos periódicamente nuestras prácticas de manejo de datos para garantizar el cumplimiento y prevenir el uso indebido.

6. Sus Derechos

  • Usted tiene derecho a saber qué datos se recopilan sobre usted y cómo se utilizan.
  • Si tiene preguntas o inquietudes sobre sus datos, puede contactarnos utilizando la información proporcionada en nuestro sitio web.

7. Cambios en Esta Política

  • Podemos actualizar estos Términos de Servicio periódicamente. Los cambios se publicarán en esta página con una fecha de vigencia revisada.

Si tiene preguntas o inquietudes sobre nuestros Términos de Servicio, contáctenos a través de la página de contacto de nuestro sitio web.
